# Enlighten me

A collection of enlightening quotes from [Yogi tea](https://www.yogiproducts.com/inspirations/). Don't abuse!

## Install

```bash
npm install -g enlighten-me
```

## Usage

```bash
enlighten-me
```

or with [npx](https://nodejs.dev/learn/the-npx-nodejs-package-runner):

```bash
npx enlighten-me
```
